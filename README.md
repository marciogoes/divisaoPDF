# 🔪 Divisor de PDF PRO v2.0

Ferramenta web completa para manipulação de arquivos PDF — 16 operações.

## 🚀 Deploy no Railway

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app)

## ✨ Funcionalidades

**Divisão:** Partes iguais · Páginas por parte · Por marcadores  
**Extração:** Páginas específicas · Intervalo · Converter para imagens  
**Edição:** Rotacionar · Remover · Reordenar · Recortar margens  
**Adicionar:** Marca d'água · Numeração de páginas · Metadados  
**Segurança:** Proteger com senha · Remover senha · Comprimir  

## ⚙️ Instalação local

```bash
py -m pip install -r requirements.txt
py app.py
```

Acesse: http://localhost:5000

## 🛠️ Tecnologias

- Python + Flask + Gunicorn
- PyPDF2 · reportlab · pymupdf
